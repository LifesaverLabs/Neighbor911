## Nomor Darurat

**Nomor Darurat Nasional:**
- **112** - Nomor Darurat Nasional
- **118** - Ambulans
- **119** - Pemadam Kebakaran
- **110** - Polisi
- **115** - SAR (Search and Rescue / Pencarian dan Penyelamatan)
- **129** - Pusat Informasi Keracunan
